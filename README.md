# Task 6: Flexbox and Grid Login Layout

## Description
This task is a responsive landing page section built with HTML and CSS. It demonstrates a two-column layout using CSS Grid for a login-style interface with visual balance between an image and form panel.

## Features
- Two-column layout using CSS Grid
- Centered form controls and login button
- Responsive spacing for form elements
- Simple `header` section with image and user login form
- Styled input fields, button, and footer link

## Files
- `index.html` - page structure and markup
- `style.css` - layout styling, grid, and responsive form style

## Usage
1. Open `index.html` in a browser.
2. The page displays a split-screen layout with:
   - left side image
   - right side login form
3. Use the text fields to enter email and password visually.

## Notes
- The form is not connected to a backend.
- The layout uses CSS Grid and flex utilities for alignment.
